# Electric Hues v1.0
Innioasis Y1 Themes
My first Innioasis Theme! Initial inspiration from mp3li's Animal Crossing theme, with additional support from Themeless and Yourtheme, as well as https://themes.innioasis.app/creators/.

Most icon designs were traced from Canva images with neon elements added by me. Background images sourced from unsplash and Pexels.
